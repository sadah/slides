What is Zen-Coding ?

Zen-Coding をすごく簡単に表現すると

  • 特定の省略記法の文字列を、HTMLに展開する
  • 様々なエディタに対応したプラグイン
  • オープンソース / MITライセンス

Zen-Coding for Textarea

Zen-Coding を Textarea で利用するサンプル

  • Coding for Textarea

他のテキストエリアでも使えたら便利だなと思って探してみると、UserScript は見つかった。

  • Zen Coding for any website for Greasemonkey

but ...

  • インデントが tab
    → スペースふたつがいい
  • HTMLタグを展開すると lang=en
    → jaがいい
  • タグを展開するショートカットが Meta + E
    → Eclipseっぽく Ctrl+Space がいい

and ...

  • HTML をゼロから書くことが少ない。(僕は)
  • 文書構造を考えたりすることも少ない。(僕は)
  • それでもちょっとしたHTMLは日常的に書く。(たぶんみんなも？)
  • blog を書いたりするとき。あるいは Wiki や CMS にちょっとしたものを書くときに。
