WHATWG Web Hypertext Application Technology Working Group 2004年に Apple、Mozilla、Opera が WHATWG 設立 ブラウザベンダーはWebアプリケーションのために、HTMLを拡張したかった。 しかし、その当時のW3Cはあまり興味を示さなかった。 Web Applications 1.0 / Web Forms 2.0 を WHATWG が策定した。
